Apakah PaaS itu?

Implementasi komputasi awan yang menyediakan sumber daya dan lingkungan pengembangan terkonfigurasi bagi pengembang perangkat lunak untuk membangun aplikasi dan layanan melalui internet

  1. Sistem operasi
  2. Lingkungan pemrograman server
  3. Sistem database
  4. Storage
  5. Jaringan intranet dan internet
  6. Utilitas design dan pengembangan
  7. Perangkat lunak pendukung server lainnya
